开博前言

发布 : 2019-05-27 分类 : 其他

为什么要开博客

中断写博客已经有好几年了，主要原因还是因为之前写博客的地方，突然就不运营了，导致有些失望。
比如在百度空间，写了很多博客，以前做Windows软件时，很多学习的过程都记录了，结果就没了。

后来换了工作后，一直都处于忙的创业状态，有时间的时候更多的选择去休息了，没有想写博客。但是学的东西平时也会在ONE NOTE上记，但始终没有写的更认真。在网上写博客，可能会有人看到，所以写的时候会谨慎一些，如果对他人有帮助了，也确实能有点开心。

这几年工作也越来越忙，年纪也越来越大，程序员的危机感在整个网络蔓延，我也想留点东西表示自己是个程序员。

为什么选择这里

开博客之前调研了一阵子，目前写技术博客多的地方就这几个：

  • CSDN
  • 博客园
  • 简书
  • 知乎
  • Github Pages
  • segmentFault
  • 稀土掘金

CSDN 和 博客园的页面一看看就很程序员，没有美感（捂嘴笑），简书知乎什么的又觉得内容乱乱的，Github Pages虽然上手要稍微学一下，但是我是程序员啊，不会就得学啊，而且用它写博客，本地肯定有仓库备份，不至于网站没了，资料都没了，安全感很重要。

于是我就抽空学的搭了一下，还挺简单，看了一些教程，挑了一个看着写的最清楚，最简单的教程，很快成功：

使用Github pages和Hexo搭建自己的独立博客【超级详细的小白教程】
我就不写过程了，直接甩地址：
https://blog.csdn.net/qq_36759224/article/details/82121420

博客得用markdown格式写，写的工具用啥呢，试了下windows带的Markdown Pad， 结果不知道为啥，预览挺正常的，但保存后的文件用别的编辑器打开，没有换行，乱七八糟，我这个主题也不能识别，果断放弃，然后打开天天用的 VSCode， 搜个markdown插件，分分钟达到满意效果，哈哈哈。

写些什么内容

写什么呢？

因为这几年一直坚持在用笔写日记，本子最开始用的趁早，今年换了个口味用的卡杰，日记里主要就是流水账，写写今天发生了什么事，等以后可以查到之前干了什么，因为肯定记不住这些琐事。

博客就主要写学习的一些事情，主要是程序方面的技术吧，把平时学习新技术点的过程记下来（怕重来一遍忘了），把遇到的一些bug和解决过程记下来（怕以后又遇上，也可能会帮助其他人解决）。

另一方面，可能也会写音乐，毕竟这也是从小到大都在学的东西，但是也会受工作时间影响，可能少点。

所以就在文章开头，先放首自己的原创歌曲《寂寞的风景》，激励自己坚持下去吧。

再就是一些生活随笔了，可能也不多吧，毕竟我不怎么说心情。

总之，新博客，我来了。
